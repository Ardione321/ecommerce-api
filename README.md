## Description

This is an e-commerce API built using the **NestJS** framework with **TypeScript**. It handles various e-commerce functionalities such as managing products, orders, user profiles, inventory, reviews, and order history.

## Features

- Product management (CRUD operations for products)
- User management (CRUD operations for user profiles)
- Order management (creating and managing orders)
- Reviews and ratings for products
- Inventory management
- Secure authentication with JWT
- And more...
